## 使用方法

| 命令 | 功能 |
|---|---|
| !!whitelist | 查看帮助 |
| !!whitelist add <玩家> | 将玩家添加至白名单 |
| !!whitelist remove <玩家> | 将玩家移出白名单 |
| !!whitelist search <玩家> | 判断玩家是否添加至白名单 |
| !!whitelist list | 列出所有在白名单内的玩家 |
| !!whitelist reload | 从白名单文件重新加载白名单 |
| !!whitelist on | 开启白名单 |
| !!whitelist off | 关闭白名单 |
